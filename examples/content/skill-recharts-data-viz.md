# Recharts Data Visualization

Interactive charting with Recharts for React applications. Covers radar charts, bar charts, sensitivity analysis visualizations, and real-time chart updates driven by state changes.

## Demonstrated Capabilities

- Radar charts for multi-dimensional option comparison (up to 6 options)
- Bar charts for sensitivity analysis weight visualization
- Real-time chart re-rendering on state change (< 50ms update cycle)
- Responsive chart sizing within desktop app layouts
- Custom color schemes and chart styling
- Data transformation from scoring engine output to chart-compatible format
