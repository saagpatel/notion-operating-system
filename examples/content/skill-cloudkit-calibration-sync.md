# Calibration Game Architecture with SwiftUI + CloudKit

Building an iOS prediction-training product with local-first app state, CloudKit sync, and long-horizon reasoning metrics.

- SwiftUI product architecture for daily prediction workflows
- SwiftData-backed local models with CloudKit sync posture
- Confidence-interval gameplay that supports long-run calibration scoring
- iOS-native entitlements and deployment posture for a zero-infrastructure backend
- Simulator build verification on a real Xcode project baseline
