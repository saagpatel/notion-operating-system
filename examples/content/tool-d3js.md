# D3.js

JavaScript library for data-driven document manipulation and visualization. Used in thought-trails for force-directed graph rendering of LLM reasoning chains with real-time node streaming.

## Usage Context

- Force-directed graph simulation with dynamic node/edge insertion
- SVG rendering with zoom, pan, and interactive selection
- Force parameter tuning for visual stability during streaming updates
- Export to SVG format for sharing and archiving
