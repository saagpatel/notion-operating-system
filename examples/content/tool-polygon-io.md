# Polygon.io Market Data API

Market data API used for live stock ticker quotes inside desktop monitoring tools.

## Key Details

- Provides real-time and delayed market data feeds for stock symbols
- Useful for lightweight ambient monitoring products where quick quote refresh matters more than deep brokerage features
- Suitable for desktop overlays that need a documented, production-grade ticker source instead of unofficial scraping
- Free-tier limitations should be treated as a product constraint during early validation
