# Transparent Click-Through macOS Overlay via Tauri

Building a macOS desktop overlay that stays always-on-top, supports transparent rendering, and switches cleanly between ambient and edit interaction modes.

## Demonstrated Capabilities

- Tauri desktop shell with a Rust backend and React renderer
- Transparent overlay behavior for ambient log and ticker monitoring
- Edit-mode transitions that preserve focus while still allowing panel management
- Local log streaming plus remote feed integration in the same desktop surface
- Persistent desktop configuration for panel state and overlay preferences
