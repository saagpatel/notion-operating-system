# Local Argument Mapping with React Flow + SQLite

Implementing a structured reasoning tool on top of a node-graph canvas while keeping persistence local and deterministic.

## Demonstrated Capabilities

- React Flow canvas with typed node and edge semantics for reasoning workflows
- Local SQLite-backed persistence for maps, node content, and structural edits
- Auto-save behavior tuned separately for movement, content edits, and graph structure
- Export support for graph artifacts without introducing a cloud backend
- Domain-specific graph editing that stays understandable for non-engineering use cases
