# Interactive Bayesian Visualization with D3 + Next.js

Designing and shipping interactive probabilistic explanations as a static web product using React for UI composition and D3 for visual math and scales.

## Demonstrated Capabilities

- Chapter-based interactive essay structure with reusable explanation shells
- Slider-driven probability models with immediate visual feedback
- Static-site delivery that keeps the experience shareable and lightweight
- React-owned UI with D3 reserved for math, scales, and visualization primitives
- Testable model logic separated from presentation so reasoning behavior can be validated directly
