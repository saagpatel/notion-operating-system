# React Flow

Node-graph UI framework used to build structured visual editing surfaces for reasoning, mapping, and workflow tools.

## Key Details

- Provides pan, zoom, node handles, selection, and edge primitives for React apps
- Well suited to local-first desktop graph tools where the product logic sits above the canvas engine
- Reduces time spent rebuilding generic graph interactions from scratch
- Best used with strong domain types so graph editing semantics stay meaningful
