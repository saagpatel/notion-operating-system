# Tauri 2.0 Desktop Development

Full-stack desktop application development using Tauri 2.0 with Rust backends and React frontends. Covers IPC command patterns, SQLite integration, native OS features (notifications, file dialogs, keychain), plugin ecosystem, and cross-platform packaging.

## Demonstrated Capabilities

- Tauri 2.0 scaffold and project setup from scratch
- Rust backend commands with specta type generation
- SQLite database integration (both tauri-plugin-sql and raw sqlx)
- Native macOS features: notifications via launchd, file picker dialogs, Keychain credential storage
- Tauri event bridge for real-time streaming (Rust → frontend)
- Plugin integration: sql, dialog, shell, notification
- Window configuration: frameless mode, minimum size enforcement, bundle identifiers
- Build and packaging for macOS distribution
