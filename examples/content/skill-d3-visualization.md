# D3.js Force-Directed Graph Visualization

Real-time interactive graph rendering using D3.js v7 force simulation. Covers live node addition during streaming, force parameter tuning, zoom/pan interactions, SVG export, and performance management at scale.

## Demonstrated Capabilities

- Force-directed graph with real-time streaming node insertion
- Force simulation tuning: alphaDecay, velocityDecay, collision radius for stability
- Interactive zoom, pan, and node selection
- Color-coded node types with visual hierarchy
- SVG and JSON export of graph state
- Performance management: 200-node cap with graceful degradation
- Collapsible subtrees and alternative layout modes (tree vs force)
- Graph search with node highlighting
