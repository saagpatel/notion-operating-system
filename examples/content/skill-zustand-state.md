# Zustand State Management

Lightweight state management with Zustand for React desktop applications. Covers store design patterns, in-memory transient state, global modal management, and view navigation state.

## Demonstrated Capabilities

- Multi-store architecture: separate stores for distinct domains (decisions, sensitivity, UI)
- In-memory transient state for sensitivity analysis (ephemeral until explicit save)
- Global modal and view navigation state management
- Keyboard shortcut integration with store-driven view switching
- Clean separation between persisted DB state and in-memory UI state
