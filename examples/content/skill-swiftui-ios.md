# SwiftUI iOS App Development

Native iOS application development with SwiftUI (iOS 17+). Covers declarative UI, Swift Charts, WidgetKit home screen widgets, StoreKit 2 in-app purchases, CoreLocation, and Keychain Services integration.

## Demonstrated Capabilities

- SwiftUI declarative layout with navigation and modal presentation
- Swift Charts: LineMark with area fill for 7-day tide curves
- WidgetKit: TimelineProvider with App Group shared data, small + medium widget sizes
- StoreKit 2: one-time IAP, dynamic price display, restore purchases flow
- CoreLocation: async location fetch, permission handling
- Keychain Services: secure API key storage and retrieval
- Strict Swift concurrency: async/await, @MainActor, no DispatchQueue
