# Recharts

React charting library built on D3. Used in TradeOffAtlas for radar charts (option profile comparison) and bar charts (sensitivity analysis weight visualization) with real-time state-driven updates.

## Usage Context

- Radar charts for multi-dimensional option scoring profiles
- Bar charts for weight sensitivity visualization
- Real-time re-rendering on Zustand state changes (< 50ms)
- Responsive sizing within Tauri desktop window constraints
