# Weekly Refresh Rollout Scorecard

## Current Replacement

Use [`weekly-notion-maintenance-operating-model.md`](./weekly-notion-maintenance-operating-model.md) as the current weekly operating guide.

## Why This Document Still Exists

- the original rollout gate criteria
- the standard that the earlier live-cutover path tried to satisfy
- why the operating model was later simplified

Do not use this scorecard as the current go/no-go gate for weekly operations.
